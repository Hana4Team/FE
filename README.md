## ✏️ 기능 소개

<p>● JWT를 이용한 회원가입 및 로그인</p>
<p>● 소비패턴 확인 및 예산 설정</p>
<p>● 머니박스, 예적금 상품가입 및 해지</p>
<p>● 100일 적금, 예적금 로드맵</p>
<p>● 네이버 API를 이용한 뉴스 조회</p>

## 프론트엔드 컨벤션

| Type | Description |
| --- | --- |
| Feat | 새로운 기능 추가 |
| Fix | 버그 수정 |
| Docs | 문서 (문서 추가, 수정, 삭제) |
| Style | 스타일 (코드 형식, 세미콜론 추가: 비즈니스 로직에 변경 없는 경우) |
| Refactor | 코드 리팩토링 |
| Test | 테스트 (테스트 코드 추가, 수정, 삭제: 비즈니스 로직에 변경 없는 경우) |

| Design | css |
